  • The first thing I noticed is how the browser displays each file type seperatley and not just as text for every one of them. For example, even though the text file and pdf file display text, they do so in different ways. The pdf opens in a pdf viewer, while the text file opens as plain text.
  • The second thing I noticed was the speed at which the image loaded compared to every other file. It makes sense that the image loads slightly slower than all the other files since it is so much larger, but that made me think about how long loading a folder of images would take for example, or loading HTML and JS files that are much larger than the one I have on this website.
  • The last thing I noticed was just how strict the browser is when it comes to handling different file types, every file has to be named correctly with the correct file extension or else the browser will not know how to handle it. And if anything in the file is not correct such as incorrect HTML tags or incorrect JSON formatting, the browser will not be able to display it correctly.
